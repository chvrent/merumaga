# メルマガ運用・開発仕様書 (SPEC.md)

## 1. プロジェクト概要
- **名称**: mail-magazine-maker
- **目的**: 社内用メルマガスケジュールの可視化と共有。
- **公開先**: GitHub Pages

## 2. データ仕様
- **データソース**: Googleスプレッドシート（GASによりJSON化）
- **データ形式 (`data.json`)**:
  ```json
  [
    {
      "date": "2024-05-01",
      "time": "10:00",
      "media": "媒体名",
      "title": "メルマガ件名",
      "target": "配信ターゲット",
      "pic": "担当者",
      "status": "進行中"
    }
  ]
  ```
- **配置**: C列以降を実データとして扱う。

## 3. 運用フロー
1. スプレッドシートを更新。
2. 「確定」ボタンでデータをGitHubへ送信（自動化予定）。
3. Webサイトが自動更新される。

## 4. 技術仕様
- **GitHub Pages**: 静的サイト公開。
- **GitHub Actions**: 自動デプロイ。
- **Google Apps Script**: データ抽出・送信。
